# 🌐 Gooy.site

> A window to clarity in the darkness of the internet.

Gooy is a curated knowledge and awareness platform focused on:
- Free software & open source ecosystem
- Security awareness and vulnerability news
- DevOps, programming, networking education
- Privacy-respecting tools and communication platforms
- Curated external knowledge from the open internet
- AI-assisted research and synthesized insights

It is designed for environments where access to information may be limited, filtered, or noisy — focusing on **signal, not noise**.

---

## 🎯 Purpose

Gooy aims to provide:

- 🧠 Reliable technical knowledge
- 🛡️ Security and vulnerability awareness
- 🧰 Free and open-source software discovery
- 🌍 Curated internet content (blogs, channels, communities)
- 🤖 AI-assisted synthesized research (verified content only)

---

## 🧱 Content Structure

Gooy uses a 3-layer taxonomy system:

- Categories → main content domains
- Subcategories → focused technical areas
- Tags → flexible metadata for filtering and discovery

---

## 🗂️ Categories

- News
- Learn
- Download
- Discover
- Deep

---

## 🧭 Content Types

### 📰 News
Short, high-signal updates from the tech world:
- Security incidents
- Open-source ecosystem updates
- Important programming or infrastructure news

### 🧠 Learn
Structured educational content:
- Programming
- DevOps
- Networking
- Security fundamentals
- Artificial Intelligence

### 📦 Download
Curated tools and software:
- Free and open-source software
- Privacy-respecting applications
- AI tools and models
- Communication tools

### 🌐 Discover
Hand-picked external resources:
- Blogs
- Websites
- Online communities
- Communication channels

### 🤖 Deep
AI-generated or AI-assisted analytical content, that is verified and curated by the author:
- Research summaries
- Technical deep-dives
- Cross-source analysis
- Structured explanations of complex topics
